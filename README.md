# Student Performance Risk Prediction (AI Project)

This project predicts whether a student is **at risk** based on academic and behavioral factors using **Logistic Regression**.

## Features Used
- Attendance percentage
- Study hours
- Previous exam marks
- Assignment completion rate

## Tech Stack
- Python 3.11
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## What This Project Demonstrates
- Supervised Machine Learning
- Data preprocessing
- Model training & evaluation
- Risk prediction
- Data visualization
